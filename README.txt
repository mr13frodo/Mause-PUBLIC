Mause 0.1



СКАЧАННЫЙ РЕПОЗИТОРИЙ ИЗМЕНЯТЬ НИ В КОЕМ СЛУЧАЕ НЕЛЬЗЯ. ДЛЯ ДАЛЬНЕЙШЕГО ИСПОЛЬЗОВАНИЯ ЕГО ЛИШЬ КОПИРОВАТЬ, ИНАЧЕ СЛОМАЕТСЯ МЕХАНИЗМ ОБНОВЛЕНИЯ



Отдельно вырезанный функционал спама по чата. Авторизация и заполнение данными происходит один раз, далее софт автоматически подтягивает все. В папке находится две версии софта:
- MauseP - написан на Python, если работает стабильно без нареканий, то рекомендуется использовать исключительно его.
- MauseC - написан на C# для случаев, когда версия на Python требует слишком много ресурсов, не рекомендуется использовать без крайней необходимости.
